# BlueCrow POS Design System

## Tokens globales
- Tipografía: Inter / Roboto / sans-serif.
- Colores: `primary`, `secondary`, `success`, `danger`, `warning`, `background`, `border`.
- Espaciado: 4, 8, 12, 16, 24, 32.
- Bordes: `--radius-md`, `--radius-lg`.
- Sombras: `--shadow-sm`, `--shadow-md`.

## Componentes UI
- `x-ui.card`
- `x-ui.button`
- `x-ui.input`
- `x-ui.select`
- `x-ui.table`
- `x-ui.form-section`
- `x-ui.breadcrumbs`
- `x-ui.page-header`

## Regla
Las vistas deben reutilizar estos componentes y evitar estilos inline.
